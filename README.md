# e-plantShopping

e-plantShopping is a React-based plant shopping application created as part of the IBM/Coursera project. The application allows users to browse different plant categories, view plant details, add plants to a shopping cart, update item quantities, remove items from the cart, and continue shopping.

## Project Overview

This project demonstrates a simple e-commerce shopping experience for plants using React and Redux state management.

## Features

- Display plants grouped by categories
- Show plant name, image, description, and cost
- Add plants to the cart
- Track cart items using Redux
- Increase or decrease item quantity
- Remove items from the cart
- View total cart cost
- Continue shopping from the cart page
